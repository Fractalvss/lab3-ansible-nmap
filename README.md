# Lab 3. Ansible - работа с переменными и циклами (NMAP)

## Задача
Создать плейбук, который:
- Устанавливает NMAP на целевые хосты
- Читает файл со списком целей
- Сканирует каждую цель с ключом `-p 80` в цикле
- Собирает и выводит результаты сканирования

## Особенности реализации
- Список целей читается через `lookup('file', ...)` на control node
- Сканирование выполняется на managed node в цикле `loop`
- Результаты сохраняются через `register` и выводятся через `debug`
- Использован флаг `-Pn` для сканирования хостов, блокирующих ICMP

## Структура проекта
- `playbook.yml` - плейбук Ansible
- `inventory.ini` - инвентарь с целевыми хостами
- `targets.txt` - список целей для сканирования

## Запуск
```bash
ansible-playbook -i inventory.ini playbook.yml
```

## Результат
Плейбук успешно устанавливает NMAP и сканирует все цели из файла targets.txt, выводя подробные результаты для каждого хоста.
